# DCF Valuation Model – TCS
DCF valuation model of TCS with 5-year projections, WACC-based discounting, and sensitivity analysis to estimate intrinsic value

## Overview

This project focuses on valuing TCS using a Discounted Cash Flow (DCF) model. The model estimates the intrinsic value of the company based on projected future cash flows and discounting them to present value.

## Objectives

* Estimate intrinsic value of TCS
* Understand valuation sensitivity to key assumptions
* Apply corporate finance principles in practice

## Key Highlights

* Projected revenue growth from **INR 2.55Tn to INR 3.78Tn (+48%)**
* Assumed **EBITDA margin of 26%** and **WACC of 10.5%**
* Estimated **Enterprise Value: INR 8.96Tn**
* Derived intrinsic value of **~INR 2,600/share**
* Identified **~21% downside vs market price**

## Valuation Approach

* Forecast period: 5 years
* Free Cash Flow (FCF) projections
* Terminal value using growth method
* Discounting using WACC

## Sensitivity Analysis

* Built a **5×5 sensitivity matrix**
* Tested impact of:

  * WACC changes
  * Terminal growth rate
* Valuation range: **INR 1,024 – INR 5,032/share**

## Tools Used

* Microsoft Excel
* Financial Modelling techniques

## Files Included

* DCF model (Excel)
* Sensitivity analysis sheet
* Summary notes

## Key Insight

The valuation suggests that TCS may be overvalued under current assumptions, highlighting the importance of sensitivity analysis in investment decision-making.

## What I Learned

* Practical application of valuation theory
* Sensitivity of valuation to small assumption changes
* Importance of transparent modelling assumptions
